# Before PCBA Approval Checklist

- [ ] BOM complete with MPN for every assembled part
- [ ] All assembled parts in stock at fab house (or ordered)
- [ ] CPL/centroid file matches BOM (every part has a placement)
- [ ] STM32 dev module excluded from PCBA (hand-soldered)
- [ ] Other excluded parts clearly documented
- [ ] Component orientations verified in fab house 3D preview
- [ ] WARNING: Bottom-side orientation UNVERIFIED -- double-check
- [ ] Assembly side(s) selected (top/bottom/both)
- [ ] PCBA remarks field filled in (excluded parts, notes)
- [ ] Engineering questions from fab house answered
- [ ] Estimated cost reviewed
- [ ] Order confirmed
