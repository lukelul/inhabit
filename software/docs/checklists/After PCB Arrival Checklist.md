# After PCB Arrival Checklist

- [ ] Count boards received vs. ordered
- [ ] Visual inspection: solder quality, component placement, silkscreen
- [ ] Check IC orientations (pin 1 markers)
- [ ] Check for missing components vs. BOM
- [ ] Check for solder bridges (magnifying glass / microscope)
- [ ] Photograph boards (top and bottom) for records
- [ ] Label boards with serial number and date received
- [ ] Store unused boards in ESD-safe packaging
- [ ] Prepare for hand-soldered parts (STM32 dev module, connectors)
- [ ] Begin [[docs/hardware/bringup/Hardware Bring-Up SOP|Hardware Bring-Up SOP]]
