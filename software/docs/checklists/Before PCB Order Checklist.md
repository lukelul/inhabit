# Before PCB Order Checklist

- [ ] DRC clean in Altium (no errors)
- [ ] Gerbers generated and verified in external viewer
- [ ] NC drill file generated and origin matches Gerbers
- [ ] Board outline correct
- [ ] Layer stackup specified
- [ ] Copper weight specified
- [ ] Surface finish selected (HASL, ENIG, etc.)
- [ ] Board thickness confirmed (1.6mm default)
- [ ] Silkscreen readable (component labels, pin 1 markers, board revision)
- [ ] Fiducials present (if needed for PCBA)
- [ ] Panelization considered (if ordering multiple boards)
- [ ] Order quantity decided
- [ ] Lead time acceptable
- [ ] Budget approved
