# Before First Power Checklist

- [ ] Visual inspection passed (no visible defects)
- [ ] Continuity check: 5V5 to GND (no short)
- [ ] Continuity check: VCC_BUS/3V3 to GND (no short)
- [ ] Continuity check: CANH to CANL (no short)
- [ ] Continuity check: SPI lines not shorted
- [ ] STM32 dev module soldered and verified
- [ ] Power supply set to 5.5V
- [ ] Current limit set (start at 100 mA, TBD for nominal)
- [ ] Multimeter ready to measure 3.3V rail
- [ ] Oscilloscope available (recommended)
- [ ] ESD precautions taken
